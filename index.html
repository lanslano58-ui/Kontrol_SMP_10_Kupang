            Backup & Pulihkan Data (Excel)

            Unduh seluruh data aplikasi (Data Guru, Petugas Piket, Jadwal Mengajar, Pengumuman, dan riwayat lainnya) sebagai satu file Excel. Simpan file ini secara berkala sebagai cadangan — jika suatu saat data perlu dipulihkan (misalnya setelah Reset Total Aplikasi atau pindah ke Google Sheets baru), tinggal Import file backup ini dan seluruh data piket & jadwal pelajaran yang pernah diinput akan otomatis kembali seperti semula, tanpa perlu input ulang manual.

            Atur Jadwal Mengajar Mingguan (Kontrol KBM)

            Atur jadwal mengajar untuk 1 minggu penuh — cukup diatur sekali per hari. Admin dapat langsung menambahkan rentang jam pelajaran (Jam Mulai s/d Jam Selesai) untuk masing-masing guru. Jadwal ini otomatis dipakai di tab Jadwal Pelajaran dan modul Kontrol KBM sesuai harinya, tanpa perlu diatur ulang setiap hari.

            Khusus mata pelajaran Pendidikan Agama, satu kelas & jam yang sama boleh diisi maksimal 2 guru (kelas paralel berdasarkan agama). Untuk mata pelajaran lain, satu kelas & jam hanya boleh diisi 1 guru/1 pelajaran.

            Menampilkan jadwal untuk hari: — dikelompokkan per kelas biar mudah dibaca seperti jadwal pelajaran di papan sekolah.
